Calculator App Issue Reproduction

To reproduce the calculator issue:

  1. Open calculator: pnpm dev (running on port 3006)
  2. Click any calculator button
  3. Check if transform: scale(0.95) sticks after release
  4. Inspect element to see computed styles

Possible Causes:

  • CSS Specificity: Hover styles with !important overriding inline styles
  • State Management: Button state getting stuck in 'pressed'
  • Event Conflict: Keyboard hook interfering with button events
  • Timing Issue: Race condition between mouse/keyboard events

Debug Steps:

  1. Check computed styles on stuck button
  2. Verify button state signal value
  3. Check if mouseup/keyup events are firing
  4. Verify CSS specificity in DevTools
