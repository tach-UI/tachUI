# tachUI in action

This section collects immersive resources—playgrounds, example galleries, and blueprints—for teams that prefer learning by doing.

## What's here

- **Interactive Playground** – edit tachUI code inline and preview the output (iframe placeholder live today, tachUI-rendered playground coming soon).
- **Component Gallery** – browse 71+ components with prop controls.
- **Modifier Lab** – experiment with modifier chains and copy the generated code.
- **Patterns Library** – opinionated layouts, navigation flows, and interaction recipes.

Use the sidebar to explore each area.
