# @tachui/symbols

tachUI bundles a curated symbol system powered by Lucide plus custom glyphs to match the platform aesthetic.

- 1,000+ icons with tree-shakable imports
- Automatic theme syncing with ColorAsset + ImageAsset
- SVG + inline canvas rendering to support blur/backdrop pipelines

```bash
pnpm add @tachui/symbols@0.9.0
```
