# @tachui/primitives

Foundation components such as stacks, text, images, scroll views, lists, spacers, and form-ready controls.

## Install

```bash
pnpm add @tachui/primitives@0.9.0
```

## Coverage

- VStack, HStack, ZStack, Spacer, Divider
- Text, Symbol, Image with loading states
- ScrollView, List (virtualized), Menu, ForEach
- Button, Toggle, Picker, DatePicker, Slider, Stepper

## Status

- 22 test files, 500+ assertions
- Layout edge-case fixes tracked in `design-docs/components.md`
- Additional components planned to hit the 71+ milestone including Label, ProgressView, and Gauge.
