# Component cheat sheet

| Component | Description | Notes |
| --- | --- | --- |
| `Text` | Typography with modifiers | 96% SwiftUI parity |
| `Button` | Action button with states | Works with `.buttonStyle` modifiers |
| `List` | Virtualized lists | Provided by `@tachui/data` |
| `NavigationStack` | Hierarchical navigation | Provided by `@tachui/navigation` |
| `TabView` | Tab-based navigation | Provided by `@tachui/navigation` |
| `Form` | Group form inputs | Provided by `@tachui/forms` |

Looking for the original exhaustive table? Head to the [Component Catalog](/components/catalog) for the full list.
